Step 1

Create a Project

Click + New Project in the top-right corner and give it a name — usually the video title.

1
Click "+ New Project"
Top-right header button. Type the project name and press Enter.
2
Sub-projects for phases
Right-click any project in the sidebar to add a child — useful for Pre-Production and Post-Production phases.
3
Mark complete when done
Right-click a project and choose Complete to move it to the Completed section at the bottom of the sidebar. Keeps your active list clean.
Step 2

Set Up Role Assignees

Open the Export Schedule panel at the bottom, then click the Role Assignees tab. Each project gets its own team.

1
Click a suggested name
Dashed chips show everyone who's ever been on a project. Click to add them to this one instantly.
2
Or type a new name
Type in the "Add name…" box and press Enter. Focus stays so you can keep adding quickly.
3
Names become draggable chips
Anyone added to a role appears as a People chip in the palette — drag them straight onto a calendar day to skip typing their name.

Drop a People chip on any calendar day or event block — the person's name is pre-filled in the panel so you only need to type the task.

Per-project, always up to date
Chips refresh every time you switch projects. People from all roles appear — role shown in parentheses so you always know who does what.
Step 3 continued

Creating & Editing Tasks

1
Drag a role chip onto a day
A panel slides up. Choose the Person from the dropdown (populated from Role Assignees + anyone previously assigned), add a task name, and set the Effort level.
2
Drag tasks to move them
Grab any task on the calendar and drop it on a new day. Hold Alt while dragging to duplicate it.
3
Resize for multi-day tasks
Drag the right edge of a task to stretch it across multiple days. The bar stays unbroken across the week.
Step 3 continued

Reviews, Shoots & Key Dates

Red chips create event blocks. Gray chips mark Key Dates. Both work differently from regular tasks.

1
Three key date types
🏁 Project Due Date — final deadline. 🎯 Milestone — internal checkpoint. 📦 Delivery — asset or deliverable hand-off. All appear as gray date markers.
2
Add attendees by dropping role chips onto the event
Drop any role or People chip onto a red block or gray Key Date to add an attendee. Their name badge appears on the block.
3
Switch event type by clicking the badge
Click the colored label badge on any event (e.g. "Review") to convert it — Review, Studio Shoot, On-Location Shoot, Project Due Date, Milestone, or Delivery.

Step 4

Exporting Your Schedule

The Export Schedule panel at the bottom has five tabs. Your last-used tab is remembered between sessions.

Text
Table / Excel
PNG
Trello Export
Role Assignees
Text — Formatted schedule ready to paste into email, Slack, or Word.
Options: Bold dates · Toggle roles · Align tasks in columns.
Step 4 continued

Export Tab Details

📝
Text
Formatted plain text. Copy writes real bold for Word/PowerPoint and ** markers for Slack/Notion simultaneously.
📊
Table / Excel
Tab-separated values. Copy as TSV, or select all in the table and paste directly into Excel.
🖼
PNG
Downloads a high-resolution image of the next 4 weeks (or longer if tasks extend beyond that).
✅
Trello Export
Copies a checklist of producer categories to paste into Trello or Planyway. Tasks are automatically routed to Pre-Production, Producing, or Post-Producing.
Step 4 continued

Role Assignees Tab

The Role Assignees tab in the Export panel is where you build the team for each project. It's separate from the export tools — think of it as project setup.

👤
Add people per role
Each role (Producer, Editor, etc.) has its own input. Type a name and press Enter — they're added to this project's roster immediately.
⚡
Suggested names
Dashed chip buttons show names from previous projects. Click any suggestion to add them instantly — no typing needed.
🎯
Roster drives the palette
Names you add here become People chips in the palette. Drag a chip onto any task to assign that person in one move.
Tips

Managing Projects

⋯
Right-click any project for options
Rename, Add Subproject, Duplicate, Complete, or Delete. Deleting a project removes all its subprojects too.
✓
Complete moves projects to a separate section
Completed projects move to a Completed section at the bottom of the sidebar, collapsed by default. Right-click to reopen or delete.
⎇
Alt + drag to duplicate
Hold Alt and drag a project in the sidebar to create a full copy — including all tasks, subprojects, and attendees.
