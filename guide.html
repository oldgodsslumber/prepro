Step 1

Create a Project

Click + New Project in the top-right corner and give it a name — usually the video title.

1
Click "+ New Project"
Top-right header button. Type the project name and press Enter.
2
Sub-projects for phases
Right-click any project in the sidebar to add a child — useful for Pre-Production and Post-Production phases.
Step 2

Set Up Role Assignees

The Role Assignees panel lives at the bottom-right. Each project gets its own team — add the people working on this video.

1
Click a suggested name
Dashed chips show everyone who's ever been on a project. Click to add them to this one instantly.
2
Or type a new name
Type in the "Add name…" box and press Enter. Focus stays so you can keep adding quickly.
3
Remove with ✕
Click ✕ on any name chip to remove them from this project. They'll still appear as a suggestion next time.
Step 3

Step 3 continued

Creating & Editing Tasks

1
Drag a role chip onto a day
A panel slides up. Choose the Person from the dropdown (populated from your Role Assignees), add a task name, and set the Effort level.
2
Drag tasks to move them
Grab any task on the calendar and drop it on a new day. Hold Alt while dragging to duplicate it.
3
Resize for multi-day tasks
Drag the right edge of a task to stretch it across multiple days. The bar stays unbroken across the week.
Step 3 continued

Reviews, Shoots & Events

Red chips create calendar event blocks — these work differently from regular tasks.

1
Drag a Review, Studio Shoot, or On-Location Shoot
Give the event a name, then resize it to span multiple days if needed.
2
Add attendees by dropping role chips onto the event
Drop any role chip onto an existing red block to add that person as an attendee. Their name badge appears on the block.

Step 4

Exporting Your Schedule

The Export Schedule panel at the bottom has four tabs. All persist your last-used tab between sessions.

Text
Table / Excel
PNG
Planyway
Text — Formatted schedule ready to paste into email, Slack, or Word.
Options: Bold dates · Toggle roles · Align tasks in columns.
Step 4 continued

Export Tab Details

📝
Text
Formatted plain text. Copy button writes real bold for Word/PowerPoint and ** markers for Slack/Notion simultaneously.
📊
Table / Excel
Tab-separated values. Copy as TSV, or select all in the table and paste directly into Excel.
🖼
PNG
Downloads a high-resolution image of the next 4 weeks (or longer if tasks extend beyond that).
✅
Planyway
Copies a checklist of categories (with effort emoji) to paste into Planyway or Trello. Reference columns show dates and assignees — they are not copied.
